Bootcamp 2020: MIDI Keyboard

After attending a lecture based on the sound library for Processsing I wanted to make something involving user interaction and sound. So I decided to make a MIDI Piano. I searched online for .wav piano note files however it was hard to find good quality ones and I don't have a Piano so I couldn't record my own. https://samplefocus.com/collections/grand-piano-single-notes This is the website I downloaded the notes from. I used the Keypressed, KeyReleased and switch functions to create the visual represnentation of the user pressing the keys. The notes on your keyboard would be S-K would be CDEFGAB. E=C# R=D# Y=F# U=G# I=A#

Check out the code on Github: https://github.com/BaboaAtCity/adbx721
